Aplikasi Pengelola Kata Sandi Pribadi (Secure Pass)
Repository ini berisi source code utama dari proyek Aplikasi Pengelola Kata Sandi Pribadi (Secure Pass) yang dikembangkan menggunakan bahasa pemrograman Go (Golang) untuk memenuhi tugas mata kuliah Algoritma Pemrograman.

Aplikasi ini dirancang berbasis Command Line Interface (CLI) untuk membantu pengguna mengelola, menyimpan, dan mencatat data kredensial akun secara terstruktur dan aman melalui terminal.

📁 Isi Repository
aplikasi.go: File source code utama yang memuat seluruh logika pemrograman, pengelolaan berkas, struktur data (struct account), serta fungsi interaktif manajemen kata sandi.

README.md: File dokumentasi awal yang memberikan ringkasan, deskripsi, dan keterangan pendukung mengenai proyek aplikasi ini.

🛠️ Fitur Utama Aplikasi
Penyimpanan data akun (Nama Layanan, Email, Kata Sandi).

Pencatatan tanggal pembaruan kata sandi secara otomatis.

Antarmuka interaktif berbasis teks yang ringan dan mudah digunakan.

Teks di atas sudah disesuaikan dengan struktur kode yang kelihatan di file aplikasi.go milikmu sebelumnya (seperti adanya komponen nama_layanan, email, kata_sandi, dan tgl_pembaruan).
